## ✨ Overview

**Notepaper** is a lightweight, no-dependency notes app that runs entirely in the browser. Create, edit, and delete notes with a clean interface — all data is saved to `localStorage` so nothing is lost on refresh.

---

## 🎯 Features

| Feature | Description |
|---|---|
| ➕ **Add Notes** | Write a note with an optional title and content |
| ✏️ **Inline Editing** | Click directly on any note to edit it on the spot |
| 🗑️ **Delete Notes** | Remove any note instantly with one click |
| 📄 **Pagination** | Navigates 5 notes per page — no endless scrolling |
| 🔢 **Note Counter** | Live badge showing total number of notes |
| 🕐 **Timestamps** | Each note displays its created date and time |
| 💬 **Toast Alerts** | Subtle notifications for add and delete actions |
| 🌑 **Empty State** | Friendly message when no notes exist |
| 💾 **Persistent Storage** | All notes saved to localStorage automatically |

## 🚀 Getting Started

No build tools. No installs. No dependencies.

```bash
# 1. Clone or download the project
git clone https://github.com/your-username/notepaper.git

# 2. Open in browser
open index.html
```

Or just download the zip and open `index.html` directly in any browser.

---

## 🖱️ How to Use

```
1. 📝  Type a title (optional) and your note content in the form
2. 💾  Click Save Note to add it to your list
3. ✏️  Click on any note's title or content to edit it inline
4. 💡  Click away (blur) to auto-save your edits
5. 🗑️  Click Delete on a note to remove it permanently
6. 📄  Use Prev / Next or page numbers to navigate
```
